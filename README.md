# 工会战报刀 BOT

基于 NoneBot2 + Python 的工会战报刀机器人，适用于单BOSS顺序推进类工会战。

## 功能列表

- ✅ 报刀 / 撤刀
- ✅ 补偿刀（尾刀后额外获得）
- ✅ 每人每天3刀限制
- ✅ 出刀进度统计与查询
- ✅ BOSS血量追踪（击杀后自动进阶）
- ✅ 预约BOSS（排队）
- ✅ 每日出刀情况图表汇总
- ✅ 自动催刀提醒（可配置时间）

## 项目结构

```
guild_war_bot/
├── README.md
├── requirements.txt
├── .env                    # NoneBot2 配置
├── bot.py                  # 入口文件
└── src/
    └── plugins/
        └── guild_war/
            ├── __init__.py
            ├── config.py       # 配置（BOSS血量、刀数等）
            ├── database.py     # SQLite 数据库操作
            ├── models.py       # 数据模型
            ├── handlers.py     # 指令处理器
            ├── scheduler.py    # 定时任务（催刀）
            └── chart.py        # 图表生成
```

## 安装与运行

```bash
pip install -r requirements.txt
# 配置 .env 文件后：
python bot.py
```

## 指令列表

| 指令 | 说明 |
|------|------|
| `报刀 <伤害>` | 记录一刀伤害 |
| `尾刀 <伤害>` | 记录击杀BOSS的最后一刀（获得补偿刀） |
| `补偿刀 <伤害>` | 使用补偿刀 |
| `撤刀` | 撤销最近一次报刀 |
| `预约` | 预约当前BOSS |
| `取消预约` | 取消预约 |
| `出刀进度` | 查看今日全员出刀情况 |
| `BOSS状态` | 查看当前BOSS血量和周目 |
| `今日汇总` | 生成图表并发送 |
| `催刀` | 手动催促未出完刀的成员（管理员） |
| `开启工会战` | 开始新一天的工会战（管理员） |
| `结束工会战` | 结束今日工会战（管理员） |
